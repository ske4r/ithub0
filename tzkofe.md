## Общее описание

### Проблема
Кофейня "Urban Brew" сталкивается с неравномерной загрузкой в течение дня. Пиковые часы (8-10 утра, 12-14 дня) создают очереди и нагрузку на персонал, mientras в промежуточные периоды заведение простаивает.

### Решение
Telegram-бот "CoffeeBoost" для стимулирования заказов в непиковое время через систему динамического ценообразования и бонусных программ.

### Основная ценность
- Для клиентов: Выгода при заказе в спокойные часы
- Для кофейни: Равномерное распределение нагрузки и увеличение выручки

# Псевдокод системы динамического ценообразования
def calculate_dynamic_price(base_price, current_hour):
    peak_hours = [8, 9, 12, 13, 17, 18]
    off_peak_hours = [10, 11, 14, 15, 16]
    
    if current_hour in peak_hours:
        return base_price * 1.1  # +10%
    elif current_hour in off_peak_hours:
        return base_price * 0.85  # -15%
    else:
        return base_price * 0.75  # -25% в глубоко непиковое время
